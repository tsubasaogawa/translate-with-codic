# translate-with-codic

## Overview

* Codic, which translates from Japanese to programming-optimized English, in your Bash client

## setup

Edit `access_token` file to use codic yourself.

## Usage

```bash
# using args
$ ./translate-with-codic.sh おいしい
delicious

# using stdin
$ echo 'おいしい' | ./translate-with-codic.sh
delicious
```
